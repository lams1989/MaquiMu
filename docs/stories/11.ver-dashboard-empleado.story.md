## Análisis Arquitectónico (Arquitecto)

### Decisiones de Diseño

**Alcance:** Full-Stack (Backend + Frontend Web) - Dashboard con KPIs.

**Backend (Spring Boot):**
*   **Endpoint:** `GET /api/dashboard/operario` (retorna KPIs).
*   **KPIs:**
    *   `maquinariaDisponible`: Count de maquinaria con estado DISPONIBLE.
    *   `contratosActivos`: Count de alquileres con estado ACTIVO.
    *   `facturasPendientes`: Count de facturas con estado PENDIENTE.

**Frontend (Angular):**
*   **Módulo:** `AdminModule`.
*   **Componente:** `EmployeeDashboardComponent` (tarjetas con KPIs).
