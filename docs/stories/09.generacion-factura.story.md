**Como:** Operario (Administrador o Agente)
**Quiero:** Generar facturas para los alquileres.
**Para:** Controlar los pagos y la contabilidad.

**Criterios de Aceptación:**
*   La factura se genera asociada a un alquiler (`alquiler_id`).
*   La factura se crea con un monto, fecha de emisión y un estado por defecto ('PENDIENTE').
*   La factura generada es visible en el módulo financiero y en el portal del cliente.
*   Puedo editar el estado de pago de una factura.
*   Puedo consultar todas las facturas del sistema.

---

## Análisis Arquitectónico (Arquitecto)

### Decisiones de Diseño

**Patrón Arquitectónico:** Arquitectura Hexagonal Multi-módulo + CQRS + Adaptador de Documentos

**Justificación:**
*   **Separación de Responsabilidades:** La lógica de generación de documentos (PDF) se aísla en un adaptador de infraestructura, desacoplando el dominio de librerías específicas (`OpenPDF`).
*   **CQRS:** Separación clara entre generar la factura (Comando) y consultarla/descargarla (Consulta).
*   **Reglas de Negocio:** Validación estricta del estado del alquiler (`FINALIZADO`) antes de facturar.

**Modelo de Datos (Entidad `Factura`):**
*   `facturaId`: Long (PK, Auto-increment).
*   `alquilerId`: Long (FK a alquileres, Unique).
*   `clienteId`: Long (FK a clientes, redundancia controlada para consultas rápidas).
*   `fechaEmision`: LocalDateTime (Not Null).
*   `montoTotal`: BigDecimal (Not Null).
*   `estadoPago`: Enum [`PENDIENTE`, `PAGADO`, `ANULADO`].
*   `rutaArchivoPdf`: String (Opcional, si se almacena en disco/S3, o se genera al vuelo).

**Estructura de Módulos (Backend):**

#### **1. Módulo Dominio (`dominio/`)**
*   **Modelo:** `Factura`, `EstadoPago`.
*   **Puerto (Salida):** `GeneradorFacturaPort` (Interfaz para generar el documento).
    *   `byte[] generarFacturaPdf(Factura factura, Alquiler alquiler, Cliente cliente, Maquinaria maquinaria)`.
*   **Puerto (Persistencia):** `FacturaRepositorio`.

#### **2. Módulo Infraestructura (`infraestructura/`)**
*   **Adaptador PDF:** `OpenPdfGeneradorFactura` (Implementa `GeneradorFacturaPort` usando **OpenPDF**).
*   **Persistencia:** `FacturaEntity`, `JpaFacturaRepository`.
*   **Controlador:** `ComandoControladorFactura` (Generar), `ConsultaControladorFactura` (Listar, Descargar).

#### **3. Módulo Aplicación (`aplicacion/`)**
*   **Comando:** `ComandoGenerarFactura`.
*   **Manejador:** `ManejadorGenerarFactura`.
    *   Verifica que `Alquiler` esté `FINALIZADO`.
    *   Calcula/Valida monto final.
    *   Crea `Factura`.
    *   Llama a `GeneradorFacturaPort` para obtener el PDF.
    *   Guarda `Factura` en BD.
    *   (Opcional) Guarda el PDF o lo retorna.
