# Historia #07: Solicitar Alquiler (Cliente)

**Estado:** Ready for Dev
**Fecha:** 2025-11-22
**Origen:** Importada de HISTORIAS_DE_USUARIO.md
**Estimación:** 5 Puntos

---

**Como:** Cliente
**Quiero:** Solicitar el alquiler de maquinaria desde el portal web o la app móvil.
**Para:** Iniciar un nuevo proyecto de forma rápida y sencilla.

**Criterios de Aceptación:**
*   Puedo seleccionar el tipo de maquinaria y las fechas deseadas.
*   El sistema recibe mi solicitud y la registra para aprobación de un agente.
*   Recibo una confirmación de mi solicitud.
*   El alquiler se crea con estado `PENDIENTE`.
*   Se calcula automáticamente el costo total basado en las fechas y tarifas.

---

## Análisis Arquitectónico (Arquitecto)

### Decisiones de Diseño

**Alcance:** Full-Stack (Backend + Frontend Web + Móvil) - Crear Alquiler.

**Modelo de Datos (Entidad `Alquiler`):**
*   `alquiler_id`: Int (PK, Auto-increment)
*   `cliente_id`: Int (FK a clientes)
*   `maquinaria_id`: Int (FK a maquinaria)
*   `usuario_id`: Int (FK a usuarios - agente que aprueba)
*   `fecha_inicio`: DateTime (Not Null)
*   `fecha_fin`: DateTime (Not Null)
*   `costo_total`: Decimal (Calculado automáticamente)
*   `estado_alquiler`: Enum [`PENDIENTE`, `APROBADO`, `ACTIVO`, `FINALIZADO`, `CANCELADO`]

**Backend (Spring Boot):**
*   **Domain:** `Alquiler` (Model), `AlquilerRepositoryPort`.
*   **Application:** `AlquilerService` (Lógica de negocio, cálculo de costos, validaciones).
*   **Infrastructure:** `AlquilerController`, `JpaAlquilerRepository`, `AlquilerEntity`.

**Frontend (Angular + Android):**
*   **Angular:** `ClientModule` - `RequestRentalComponent` (formulario de solicitud).
*   **Android:** `ui.rental` - `RequestRentalActivity`.
*   **Servicios:** `AlquilerService` (Cliente HTTP).

### Especificaciones Técnicas

**API Endpoints:**
*   `POST /api/alquileres`: Crear nueva solicitud de alquiler.
*   `GET /api/maquinarias/disponibles`: Listar maquinaria disponible para fechas específicas.

**Validaciones:**
*   Fecha fin > Fecha inicio.
*   Maquinaria disponible en el rango de fechas.
*   Cliente autenticado.
