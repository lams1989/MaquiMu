**Fecha:** 2025-11-22
**Origen:** Importada de HISTORIAS_DE_USUARIO.md
**Estimación:** 3 Puntos

**Como:** Cliente
**Quiero:** Consultar mi historial de alquileres (web y móvil).
**Para:** Tener trazabilidad de mis servicios contratados.

**Criterios de Aceptación:**
*   Veo una lista de mis alquileres (activos e historial).
*   Puedo ver los detalles de un alquiler específico.
*   La información incluye: maquinaria, fechas, costo, estado.

## Análisis Arquitectónico (Arquitecto)

### Decisiones de Diseño

**Alcance:** Full-Stack (Backend + Frontend Web + Móvil) - Solo Lectura.

**Backend (Spring Boot):**
*   **Endpoint:** `GET /api/alquileres/mis-alquileres` (filtrado por cliente autenticado).
*   **Endpoint:** `GET /api/alquileres/{id}` (detalle de alquiler).

**Frontend (Angular + Android):**
*   **Angular:** `ClientModule` - `MyRentalsComponent` (lista + modal de detalle).
*   **Android:** `ui.rental` - `MyRentalsActivity`.
