# Quality Assurance

Este directorio contiene los resultados de revisiones de calidad y gates de aprobación para las historias de usuario.

## Estructura

```
qa/
└── gates/              # Gates de calidad por historia
    └── [epic].[story]-[slug].yml
```

## Uso

Cuando una historia pasa por el proceso de revisión (Peer Review), los resultados se documentan aquí siguiendo el formato definido en el Método Luis.
