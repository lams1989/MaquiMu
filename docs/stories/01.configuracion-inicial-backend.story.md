## Análisis Arquitectónico (Arquitecto)

### Decisiones de Diseño

**Patrón Arquitectónico:** Arquitectura Hexagonal (Puertos y Adaptadores)
*   **Justificación:** Aísla el núcleo de negocio de frameworks y BD, facilitando testing y mantenimiento. Alineado con la documentación de arquitectura del proyecto.

**Stack Tecnológico:**
*   **Lenguaje:** Java 17
*   **Framework:** Spring Boot 3
*   **Build Tool:** **Gradle** (Solicitado por usuario)
*   **Librerías Clave:**
    *   `spring-boot-starter-web` (API REST)
    *   `spring-boot-starter-data-jpa` (Persistencia)
    *   `spring-boot-starter-security` (Seguridad)
    *   `mysql-connector-j` (Driver BD)
    *   `lombok` (Reducción de boilerplate - Confirmado por usuario)
    *   `jjwt-api`, `jjwt-impl`, `jjwt-jackson` (Manejo de JWT)

**Estructura de Paquetes:**
`com.maquimu.backend`
*   `domain`: NÚCLEO. Contiene `model` (Entidades puras), `port.in` (Interfaces Servicios), `port.out` (Interfaces Repositorios).
*   `application`: `service` (Implementación casos de uso), `dto`.
*   `infrastructure`: `adapter.in` (Controllers), `adapter.out` (JPA Entities & Repositories), `config` (Beans, Security).

### Especificaciones Técnicas

**Configuración Base de Datos (Local):**
*   **URL:** `jdbc:mysql://localhost:3306/maquimu_db`
*   **Username:** `root`
*   **Password:** (vacío)
*   **Driver:** `com.mysql.cj.jdbc.Driver`
*   **JPA Strategy:** `validate` (Recomendado ya que existe script SQL) o `update`.

**Seguridad:**
*   Implementar `SecurityFilterChain` básico.
*   Configurar CORS para permitir peticiones desde `localhost:4200` (Angular).
