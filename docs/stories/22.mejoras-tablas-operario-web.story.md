## Descripción

**Como** operario del sistema MaquiMu,  
**quiero** ver información más completa en las tablas de gestión,  
**para** identificar rápidamente a los clientes sin necesidad de navegar a otros módulos.

## Cambios Solicitados

### 1. Módulo de Clientes — Agregar columna "ID"
- Agregar una columna **ID** al inicio de la tabla de clientes en el tab "Clientes".
- Muestra el campo `clienteId` de cada registro.
- Actualizar el `colspan` del estado vacío para reflejar la nueva columna.

### 2. Módulo de Alquileres — Agregar nombre del cliente
- En la tabla de alquileres, al lado de la columna **Cliente ID**, agregar una columna **Cliente** con el nombre completo del cliente.
- El nombre se obtiene cruzando `clienteId` del alquiler con la lista de clientes cargada desde `ClienteService`.
- Mostrar formato: "Nombre Apellido" (si tiene apellido) o solo "Nombre".
- Actualizar el modal de detalle para incluir el nombre del cliente.

## Criterios de Aceptación

- [x] La tabla de clientes muestra la columna ID como primera columna.
- [x] La tabla de alquileres muestra el nombre del cliente junto al ID del cliente.
- [x] El modal de detalle de alquiler muestra el nombre del cliente.
- [x] No hay errores de compilación en el frontend.

## Archivos Afectados

| Archivo | Cambio |
|---------|--------|
| `clients.component.html` | Agregar columna ID + ajustar colspan |
| `admin-rentals.component.ts` | Inyectar ClienteService, cargar mapa de clientes |
| `admin-rentals.component.html` | Agregar columna Cliente, actualizar detalle |

## Notas Técnicas

- El backend no incluye el nombre del cliente en la respuesta de alquileres (solo `clienteId`), por lo tanto se resuelve en frontend con un mapa `clienteId → nombre`.
- Se usa el servicio `ClienteService.getClientes()` que ya existe para cargar los nombres.

### Cambios Realizados

#### `clients.component.html`
- Columna **ID** agregada como primera columna del `<thead>` mostrando `clienteId`.
- Fila de datos: celda `{{ cliente.clienteId }}` insertada antes de la celda del nombre.

#### `admin-rentals.component.ts`
- Importación de `Cliente` (model) y `ClienteService` (service).
- Inyección de `ClienteService` en el constructor.
- Propiedad `clientesMap: Map<number, string>` para mapear ID → nombre completo.
- Método `cargarClientes()` invocado en `ngOnInit()`: construye el mapa con formato `"Nombre Apellido"`.
- Método `getClienteNombre(clienteId: number): string` para resolver el nombre en la tabla y el modal.

#### `admin-rentals.component.html`
- Columna **Cliente** agregada en `<thead>` después de "Cliente ID".
- Celda `{{ getClienteNombre(alquiler.clienteId) }}` en cada fila.
- Modal de detalle: línea `<p><strong>Cliente:</strong> {{ getClienteNombre(alquilerDetalle.clienteId) }}</p>` agregada.
